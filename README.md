# InfilooCast-Bot (24/7 Free Discord Bot hosting!)

InfilooCast Bot is a Discord 24/7 Music bot from Germany!

Instalation Guide (Free 24/7)

1. Create a Application here: https://discord.com/developers/applications
2. Get your Bot token!
3. Create an account on https://wispbyte.com/a?ref=user_jo8hoc <- You can google Wispbyte or if you want to support me visit this Link!
4. Create a new Server on Wispbyte and select Python as Language!
5. Upload all files (main.py and requirements.txt)
6. Set the startup command to python3 -m pip install -r requirements.txt && python main.py
7. Start the bot!
8. After the first start remove python3 -m pip install -r requirements.txt from the startup command!

9. Thats it! Now you have the InfilooCast bot sucessfully running 24/7
